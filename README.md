# Ankita Kumari — Portfolio

A premium, product-style portfolio built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

Inspired by Linear, Stripe, and Apple — adapted for a Backend & Cloud Engineer profile.

## Features

- Cinematic hero with large editorial typography
- Fixed minimal navigation
- Full-screen project case studies (Behance-style)
- MacBook / browser / phone mockups
- Interactive skill cards
- Experience timeline
- Styled résumé viewer
- Research publication showcase
- GitHub activity section
- Labs & blog sections
- Parallax background with abstract graphics
- Smooth scroll animations throughout

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize

Edit `src/data/portfolio.ts` to update:

- Personal info & contact links
- Projects & case study content
- Skills, experience, education
- Labs, blog posts

Update colors in `src/app/globals.css`.

## Deploy

Deploy to [Vercel](https://vercel.com) with one click:

```bash
npm run build
```

## Tech Stack

- Next.js 16 (App Router)
